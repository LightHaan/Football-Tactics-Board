# 足球战术板

一个自动播放的简化足球模拟。打开后会自动开始 10 分钟一局的比赛，显示双方 11 名球员、足球、比分、时间和事件提示。

## 当前版本

- 战术板风格球场背景
- 双方 11 人，包括守门员
- 队名、队标文字、阵型和球员属性占位
- 自动传球、盘带、射门、抢断、抄截和门将扑救
- 比赛中手动换人，自动换人保留，每队最多 5 次
- 犯规后的任意球、点球、黄牌、红牌和两黄变红
- 每局 10 分钟，结束后记录上一局比分并自动重开

## 方便后续扩展的位置

- `src/data.js`：队名、队标、阵型、球员姓名号码位置、速度传球射门盘带防守等属性
- `src/simulation.js`：比赛逻辑、战术行为、随机决策、门将和射门结算
- `src/render.js`：球场、球员、足球和动画渲染
- `src/app.js`：页面启动、比分和状态显示

## 本地预览

这个项目使用浏览器模块文件，建议通过本地预览地址打开，而不是直接双击 `index.html`。
